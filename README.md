# epic-cli

```
usage: app.py [-h] -p PREFIX -j JOBNO [-u] [-f]

EPIC CLI is a command line interface for file collection

options:
  -h, --help            show this help message and exit
  -p, --prefix PREFIX   Prefix of the project
  -j, --jobno JOBNO     Job Number of the project
  -u, --upload          Upload
  -f, --filecollection  File Collection
```
